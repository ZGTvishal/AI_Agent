This Project is a learning project for building basic knowledge of working AI Agents. 
This Agent is built to help with basic reseach tasks.
I have used OpenAI, Anthropic API, Wikipedia and DuckDuckgo search to fuel the Agent. 

To install the requirements I have creared a requirment.txt file and ran it with pip command in terminal to insall everything all at once. 


To get this Agent working you will need an .env file with OpenAI/Anthropic API key-->
Ex - OPENAI_API_KEY = "********************"
